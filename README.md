Battle Ship Odin
This project is part of the Odin Project, focusing on building a Battleship game using JavaScript, HTML, and CSS.

Features
User Interface: Simple and intuitive interface to place ships and attack the opponent.
Game Logic: Implemented game rules including ship placement, turn-based attacks, and win conditions.
Animations: CSS animations for visual feedback on hits and misses.

What I Learned
JavaScript: Deepened understanding of DOM manipulation, event handling, and game logic implementation.
HTML & CSS: Enhanced skills in structuring web pages and styling elements for better user experience.
Project Management: Experience in organizing code, version control with Git, and using GitHub for collaboration.


How to Use:
Clone the Repository:
git clone https://github.com/jonjon231564/battle-ship-odin.git
Navigate to the Project Directory:
cd battle-ship-odin
Open index.html in Your Browser to start the game.
